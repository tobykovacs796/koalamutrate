This folder contains the output files from the mutation rate pipeline folders 1-5.


	data_denovo_$SAMPLE_ID$.tab -> contains the specific mutations identified in each offspring
	denovo.txt -> number of de novo mutations identified in each trio
	callability.txt -> total callable genome size for each trio
	depth.txt -> the average depth in each trio
	fnr.txt -> false-negative rate
	mutation_rate.txt -> mutation rate for each trio
	pedigree.ped -> pedigree of each trio
	Phascolarctos_cinereus.relatedness2 -> the output relatedness values between all individuals
	Plotting_mutation_rate.Rmd -> R-script used to plot the koala mutation rate with other mammal mutation rates, as well as the distribution of mutations of different classes
